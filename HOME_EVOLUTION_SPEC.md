# 🏠 AIMNIS Home v1.7: 인터랙티브 워크스페이스 구현 명세

## 1. 초기 UI 상태 (Initial State)
- `홈화면.jpg`의 기존 레이아웃을 100% 유지한다.
- 중앙 Stitch 입력창 내부에 3가지 전략 시나리오 칩(에너지, 제조, 시티)을 배치한다.

## 2. 상태 전환 로직 (The Transition)
- 시나리오 칩 클릭 시 `isWorking` 상태를 `true`로 변경한다.
- **애니메이션:** 히어로 섹션은 상단으로 압축되고, 하단 솔루션 카드는 아래로 사라지며 좌(채팅)/우(설계도) 분할 레이아웃이 나타난다. (Framer Motion 적용)

## 3. AI 아키텍트 인터뷰 엔진 (Max 3 Turns)
- **제약 조건:** 질문은 무조건 한 번에 1개, 총 3회로 제한한다.
- **종료 트리거:** 3회 답변 완료 시 채팅창 하단에 `[Create Harness]` 버튼을 크고 화려하게(Neon Blue) 노출한다.
- **유연성:** 사용자가 "대충 해줘"라고 하면 남은 질문을 건너뛰고 즉시 설계를 확정한다.

## 4. 라이브 블루프린트 뷰어 (Right Panel)
- 텍스트 위주가 아닌 **'모듈형 카드 UI'**를 사용한다.
- 사용자가 "CCTV 16개"라고 말하면 `Widgets` 카드에 즉시 반영되는 시각적 동기화를 구현한다.
- 8pt 그리드와 20px 블러가 적용된 다크 테마를 고수한다.
